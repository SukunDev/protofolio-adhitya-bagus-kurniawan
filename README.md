## Installasi

### Cloning

```bash
git clone https://github.com/SukunDev/protofolio-adhitya-bagus-kurniawan.git
cd protofolio-adhitya-bagus-kurniawan
```

### Install depedency

```bash
npm install
```

### Migration

```bash
php artisan migrate
php artisan migrate:fresh --seed
```

### Running

```bash
npm run dev

php atisan serve
```
